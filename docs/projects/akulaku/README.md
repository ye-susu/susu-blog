# Akulaku UI 3.0 改版
<ReadingTime/>
## 项目背景
2021年 Akulaku 总 GMV 新增 86%，伴随业务发展当前产品已不能满足业务诉求，如分期场景缺乏/金融业务偶合/活动缺少曝光等。所以需要对一级页面进行重新梳理与定位，打造适合业务的新产品形态。

## 搜集问题
除自身业务诉求外，我们还通过部门访谈、用户反馈、竞品分析等方式搜集了更多产品问题，来寻求设计机会点。

| 🧕🏻 用户反馈                                                     | 🙋🏻‍♂️ 自身诉求                                                     | 📋 竞品分析                                                     |
| :----------------------------------------------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| 视觉满意度一般 <br> <br> 借款产品是否安全？<br/> <br> Service页面较少浏览 <br/> <br> 品类入口比较难找 <br/> <br> 商品是否包邮？<br/> <br> 物流速度可以快一些 <br/> <br> 积分的用途是什么？ | 分期业务线下覆盖率低 <br/> <br> 线上商品及服务比较匮乏 <br/> <br> 自营贷款和贷超差异小 <br/> <br> 现金业务定位上有偶合 <br/> <br> 站内活动曝光少 <br/> <br> 转化较低的模块长期未优化 <br/> <br> 当前版本设计陈旧 | 电商业务无特色频道 <br/> <br> 新客营销弱，缺少差异化 <br/> <br> 首页引导效率差 <br/> <br> 本地化设计不足 <br/> <br> 品牌渗透弱 |

面对众多问题，我们从业务、产品、设计三个纬度进行了整理并定义了优先级，希望从不同角度找到解决方法。

| 📊 业务                                                         | 🛍 产品                                                         | 🎨 设计                                                         |
| :----------------------------------------------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| 电商业务无特色模块，无法展现平台优势 <br> <br> 分期业务线下覆盖率低，线上商品和服务比较匮乏 <br/> <br> 借款业务营销弱，自营贷款和贷超差异小 | 产品定位上出现业务耦合需要重新梳理 <br/> <br> 转化较低的模块长期未优化，拉新促活活动曝光少 <br/> <br> 针对不同用户提升首页引导效率 | 当前版本设计陈旧，品质感不高 <br/> <br> 据头部产品品牌感弱，用户品牌感知不强烈 <br/> <br> 本地化设计不足，对当地用户喜好了解不深入 |

## 制定目标

| 产品目标                                                     | 设计目标                                                     |
| :----------------------------------------------------------- | :----------------------------------------------------------- |
| 1️⃣ 满足业务诉求提升GMV。 <br> <br> 2️⃣ 梳理产品精准定位业务。 <br> <br> 3️⃣ 优化各模块，提升转化率。 | 1️⃣ 升级设计语言，提升产品视觉体验。 <br> <br> 2️⃣ 强化品牌基因，提升品牌感知度。<br> <br> 3️⃣ 本地化探索，提升本地化设计。 |

## 改版策略
为满足业务诉求，我们首先对一级页面进行了重新定位。分期业务从金融页中独立出来成为分期页，主要构建分期场景，解决业务偶合的问题。账单等用户向功能移植到个人中心，契合用户认知，放便查询。三个 tab 对应三种核心业务，产品定位更清晰。

|   菜单   | Akulaku                                                      | Paylater                                                     | Financial                                                    | Account                                                      |
| :------: | :----------------------------------------------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **内容** | **电商频道** <br> <br> 搜索 <br>服务入口 <br>额度查询<br>个性化推荐 | **分期产品**<br/><br>分期场景<br/>支付工具<br/>生活服务<br/>积分兑换 | **金融频道**<br/><br>借贷<br/>转账汇款<br/>银行开户<br/>理财保险 | **个人中心**<br/><br>订单记录<br/>资产记录<br/>权益积分<br/>服务设置 |
| **定位** | **电商业务** <br/><br>聚合商品频道、营销活动、常用服务分流。     | **分期业务** <br/><br>孵化新的分期产品、主打分期场景、支付获得积分，积分兑换优惠、促活留存。 | **金融业务** <br/><br>为用户提供借贷、存款、理财等综合金融服务。 | **个人中心** <br/><br>用户信息记录，方便用户查询、活动曝光。     |

#### 二、强化品牌基因

##### 1. 增加品牌漏出
梳理品牌漏出场景，下拉刷新、加载中放弃默认样式，进行品牌化处理。首页 Tab 图标使用品牌 Logo。

##### 2. 品牌 IP 使用
IP 在品牌传播中更具穿透力，某些特殊触点可使用 IP 形象，确保品牌调性的同时也可提升产品好感度。

![](https://image.yesusu.top/PicGo/ui3.0-brandexplore.png)

#### 三、本地化探索
##### 1. 金刚区图标

国内：多同色系、一致性强、造型统一。

印尼：色彩丰富、装饰性强、插画风格。

3.0 图标：更多色彩、强识别性、插画风格。

![](https://image.yesusu.top/PicGo/ui3.0-slocalization-1.png)

##### 2. 缺醒页插画

国内：色彩单一，饱和度低 - 突出场景 - 传递状态。

印尼：色彩丰富，饱和度高 - 突出人物 - 传递情绪。

我们在后续设计中也遵循了本地化特征，使用了丰富的高饱和色彩，突出 IP 形象传递用户情绪。

![](https://image.yesusu.top/PicGo/ui3.0-slocalization-2.png)

##### 3. banner 图片

国内：色彩丰富 - 装饰性强 - 渲染氛围。

印尼： 单色背景 - 板式简洁 - 突出信息。

在后续设计中，我们遵循本地化特征，使用了单色背景、简洁的板式，突出信息。

![](https://image.yesusu.top/PicGo/ui3.0-slocalization-3.png)

## 数据反馈
**Akulaku**：首页整体点击率提升 **4.25%**，电商渗透率提升 **4.81%**。

**Akulaku Paylater**：线下支付覆盖率提升 **2.45%**，用户成交 GMV **30W$**。

**Financial**：整体点击率提升 **4.25%**，现金贷点击率提升 **10.63%**。

**Account**：整体点击率提升 **8.75%**，活动区点击率提升 **5.04%**。
